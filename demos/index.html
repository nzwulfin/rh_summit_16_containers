The JS player has some apparent issues with browsers other than Firefox. If you see a repeatedly blinking hypnotic circle, you may want to try another browser or download the JSON files and use the terminal `asciinema` tool for playback.

The order below is the right order (!) there may or may not have been an out of order issue during the live session.

  1. Using ansible with a container target

    Here we run a pair of containers, and use playbook to install and do general configuration for MariaDB and Wordpress.

  2. Create new framework base containers from ansible configured bases

    Here we look at the configured containers and `docker commit` new bases for later use in our applications.

  3. Show WP tables in new MariaDB container

    4. Just a proof point that shows the specific Wordpress tables in a new application container that we built using the new framework level base images.

  4. Create a kubernetes cluster on 4 atomic hosts

    This is the orchestrated build of our container orchestration engine, using kubernetes, docker, and flanneld (for overlay networking). This uses the kubernetes/contrib ansible playbooks

  5. Check cluster health and launch atomic app

    A proof point showing the use of the `atomic run` support for Dockerfile LABELs to launch an existing app from the Docker Hub.
